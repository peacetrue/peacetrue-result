= 响应数据规范

响应数据规范定义了响应数据的 xref:standard/structure.adoc[标准结构] 以及不同响应状态下响应数据的具体内容。

响应状态和异常是相对的，为了弄清楚不同的响应状态，
我们就需要分析清楚整个请求响应过程中可能出现的异常。
